# Kitty Configuration

A highly optimized configuration for the **Kitty** terminal emulator focusing on rendering performance and typography.

## Features

- **Typography**:
  - Font: `JetBrainsMono Nerd Font` (size 11.0).
  - Ligatures: Always enabled.
- **Rendering & Speed**:
  - Fine-tuned input and repaint intervals (`repaint_delay 10`, `input_delay 2`) for fast input response.
  - Audio bell disabled.
- **Visual Styles**:
  - Background opacity set to `0.90` with dynamic adjustment enabled.
  - Window decorations hidden for a borderless aesthetic.
  - Default shell set to `zsh`.
- **Interactive FX**:
  - Beam cursor style.
  - **Cursor Trails**: Enabled a trail effect (`cursor_trail 10`) for quick cursor tracking.
  - Underline URL detection with customizable link highlight styles.
