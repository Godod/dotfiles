# Fastfetch Configuration

A customized, clean system fetch configuration mapping system details and hardware specifications inside modern ascii borders.

## Layout Details

- **Visual Accent**: Uses an Arch Linux icon colored in Cyan and Purple, matching the dynamic color themes.
- **Top Block**:
  - Operating System details.
  - Hardware Device/Host name.
  - Linux Kernel version.
  - Installed packages count.
  - Active terminal emulator.
  - Window Manager.
- **Bottom Block**:
  - CPU model and specifications.
  - GPU model and specifications.
  - RAM memory usage and limits.
  - Display resolution and refresh rate.
  - **OS Age**: Dynamic script counting days since the OS installation (determined via `/` directory creation date).
  - System Uptime.
- **Theme Accents**: Color circle display to visualize system colors.

## Usage

Simply run:
```bash
fastfetch
```
Or use the fish alias:
```bash
btw
```
