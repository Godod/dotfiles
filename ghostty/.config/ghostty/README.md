# Ghostty Configuration

A clean configuration for the modern, fast GPU-accelerated terminal emulator **Ghostty**.

## Features

- **Typography**:
  - Primary font: `Google Sans Code`.
  - Secondary fallback: `0xProto Nerd Font Propo`.
  - Font size: `12`.
- **Aesthetics**:
  - Background opacity set to `0.95` for subtle transparency.
- **Dynamic Theming**:
  - Integrated with `matugen`. Colors are generated from the current wallpaper and written to `themes/Matugen` which is loaded as the active theme.
- **Convenience**: Hides the mouse pointer while typing.
